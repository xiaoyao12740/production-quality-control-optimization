**# Production Quality Inspection and Rework Decision Optimization

基于 MATLAB 的产品装配、质量检测、拆解返工与利润优化决策项目。

## 项目简介

本项目围绕制造系统中的质量检测与利润优化问题展开，研究在零部件采购、检验、装配、成品检测、售后退换和拆解返工等环节共同作用下，企业应如何制定最优决策策略，以实现利润最大化或风险控制下的稳健收益。

项目使用 MATLAB 对不同阶段的质量决策问题进行建模，分为四个子问题：

- **q1：抽样检验方案设计**
- **q2：双零件—单成品阶段检测/拆解决策**
- **q3：多零件—半成品—成品多阶段利润优化**
- **q4：风险与稳健决策分析**

项目兼具统计检验、策略穷举、利润分析和稳健优化等特点，适合作为 MATLAB 运筹优化、生产决策和质量管理方向的综合性课程/建模项目。

## 研究内容

### 1. 抽样检验方案设计（q1）

在实际生产中，企业往往需要先通过抽样检验来判断一批产品是否可接受。  
本项目在 q1 中基于：

- 原假设次品率 `p0`
- 一类错误率 `alpha`
- 二类错误率 `beta`

构建了 **SPRT（序贯概率比检验）** 方案，并自动生成：

- 接收/拒收边界
- OC 曲线
- ASN 曲线
- `p1` 灵敏度分析图
- 对数似然路径图
- 检验总结表（Excel）

这部分体现了“统计质量控制”在生产决策前端的应用。

### 2. 双零件—单成品阶段利润分析（q2）

在 q2 中，项目考虑一个较简化的装配系统：

- 两个零件
- 一个成品阶段
- 可选择是否对零件进行预检
- 可选择是否对成品终检
- 可选择是否对不合格品拆解

程序将策略写成二进制决策变量：

- `I1`：是否检验零件1
- `I2`：是否检验零件2
- `IF`：是否检验成品
- `D`：是否拆解不合格品

于是共形成 `2^4 = 16` 种策略。程序对不同情形分别计算单位利润，并输出：

- 各策略利润排序
- Top 策略
- 情形对比图
- 灵敏度分析
- Excel 汇总表

### 3. 多阶段装配与返工决策（q3）

在 q3 中，问题进一步扩展到更复杂的生产系统，包括：

- 多个零部件
- 多个半成品阶段
- 成品装配阶段
- 半成品检测与拆解
- 成品检测与拆解
- 售后换货损失与返工成本

相比 q2，这部分更加接近真实制造流程。程序通过大量策略组合穷举评估不同决策的利润表现，并将结果导出到 Excel 文件中。

### 4. 风险与稳健优化（q4）

q4 在前面利润模型基础上进一步加入不确定性和风险考虑。

程序读取抽样信息后，基于 Beta 分布先验和单侧置信分位构造“悲观点”参数，然后重新评估 q2 和 q3 中各策略的利润。输出内容包括：

- q2 稳健对照结果
- q3 稳健对照结果
- Top 策略图
- 风险场景下的利润比较图

这部分体现了“在不确定质量水平下进行稳健决策”的思路。

## 方法说明

本项目主要使用了以下方法和思想：

- **SPRT（序贯概率比检验）**
  - 用于设计抽样检验的接收/拒收规则
  - 分析抽样成本与误判风险

- **穷举搜索**
  - 将检测、终检、拆解等决策写成 0/1 变量
  - 对所有策略组合进行逐一评估

- **利润分解**
  - 将总利润拆分为：
    - 采购成本
    - 检测成本
    - 装配成本
    - 拆解成本
    - 售后退换损失
    - 销售收益

- **敏感性分析**
  - 分析关键参数变化对最优策略和利润的影响

- **稳健分析 / 悲观点分析**
  - 通过 Beta 分布分位构造偏保守参数
  - 比较风险场景下策略排名是否发生变化

## 运行环境

- MATLAB
- Excel（用于查看和保存结果表）

## 使用方法

1. 将项目文件夹保持当前目录结构；
2. 在 MATLAB 中打开项目根目录；
3. 根据需要进入对应子文件夹运行主程序：

   - `q1/main1.m`
   - `q2/main2.m`
   - `q3/main3.m`
   - `q4/main4_1.m`
   - `q4/main4_2.m`
   - `q4/main4_3.m`

4. 程序运行后会：
   - 在命令行输出关键结果；
   - 生成图像文件；
   - 将主要结果保存到 Excel 文件中。

## 输出内容

运行后可获得的典型输出包括：

- 抽样检验接收/拒收边界图
- OC 曲线与 ASN 曲线
- 策略利润排序图
- 各情形下的最优策略
- q2/q3/q4 结果汇总 Excel 表
- 风险悲观点下的稳健利润比较图

## 项目特点

- 将生产质量管理问题转化为可计算的利润优化问题
- 同时涵盖抽样检验、装配决策、返工拆解和售后损失
- 既考虑确定性利润最大化，也考虑风险场景下的稳健性
- 输出结果较完整，包含图表和 Excel 结果文件
- 适合作为 MATLAB 在质量管理、生产运筹和决策分析中的综合案例

## 局限性说明

- 当前模型主要基于题设或给定参数，不一定能直接对应真实工厂场景
- q2、q3 中很多策略依赖穷举，规模继续扩大时计算量会迅速增加
- 风险分析采用悲观点分位法，尚未扩展到更复杂的随机优化或贝叶斯决策框架
- 项目更偏重建模与策略分析，不是实际工业 MES/ERP 系统软件
